About

Source code for this site can be found at: https://github.com/mattmakesmaps/PntTrax

Some of the technologies that power this site include:

Fiona

Used to parse uploaded shapefiles, presenting a user with fields to map against database tables.

GeoDjango

A web framework used as the foundation this application.

Icons

Icons provided by the awesome Project Noun (CC-BY-SA).

Leaflet

Provides client-side mapping functionality.

Spatialite

The underlying spatial RDBMS.

Twitter Bootstrap

The base styling for all elements in this page.

Vectorformats

Created by Chris Schmidt, this library is used to provide basic conversion between Django QuerySets to output formats such as GeoJSON and KML. Accessible via service URL endpoints.
